# Guida Conformità ISO 27001

Conformità Sistema Gestione Sicurezza Informatica con Wazuh.

---

## Stato

⚠️ **Nessun modulo built-in ancora** ([richiesta feature #26711](https://github.com/wazuh/wazuh/issues/26711))

---

## Soluzione Alternativa: Implementazione SCA Personalizzata

**Passo 1**: Crea politica SCA personalizzata
```yaml
policy:
  id: "iso_27001_mapping"
  title: "Controlli ISO 27001"
  requirement: "ISO 27001:2022"

  check:
    - id: "a.5.1.1"
      title: "Politiche per sicurezza informatica"
      compliant: "true"
```

**Passo 2**: Collegamento a campo conformità Wazuh
```xml
<rule id="27001_control_verified">
  <if_sid>27001_audit</if_sid>
  <compliance>iso_27001</compliance>
  <status>true</status>
</rule>
```

**Passo 3**: Genera report con tag conformità
