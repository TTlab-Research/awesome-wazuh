# Guida Conformità GDPR

Conformità General Data Protection Regulation con Wazuh.

---

## Principi Chiave

- **Liceità, correttezza, trasparenza**: Log accesso a tutti i dati
- **Limitazione scopo**: Raccogli solo dati necessari
- **Minimizzazione dati**: Politiche di conservazione
- **Accuratezza**: Integrità audit trail
- **Limitazione archiviazione**: Cleanup automatizzato
- **Integrità & confidenzialità**: Crittografia & controllo accesso
- **Responsabilità**: Evidenza conformità

## Implementazione

```
1. Abilita politiche SCA protezione dati
2. Implementa file integrity monitoring per dati
3. Configura conservazione log basata su residenza dati
4. Audit accesso a dati personali
5. Mantieni record consenso
6. Prepara evidenza per audit
```

---

## Moduli Critici

- Monitoraggio Integrità File (protezione PII)
- Analisi Log (audit accesso)
- Risposta Attiva (remediation automatizzata)
- Reporting dashboard (evidenza audit)
