# Guida Integrazione Microsoft Teams

Invia gli avvisi Wazuh ai canali Microsoft Teams.

---

## Panoramica

Il supporto di Microsoft Teams richiede di usare Shuffle SOAR o integrazione personalizzata tramite webhook JSON.

---

## Opzione 1: Via Shuffle SOAR

Usa Shuffle per inoltrare gli avvisi Wazuh a Teams:

1. Configura istanza Shuffle
2. Crea webhook Teams
3. Costruisci workflow in Shuffle
4. Integra con Wazuh

---

## Opzione 2: Webhook Personalizzato

Crea integrazione personalizzata con webhook in ingresso Teams.
