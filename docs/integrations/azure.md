# Guida Integrazione Azure

Integra Wazuh con i servizi Azure per monitoraggio della sicurezza cloud.

---

## Servizi Supportati

- **Log Analytics**: Logging centralizzato
- **Microsoft Graph**: Monitoraggio utenti e dispositivi
- **Intune**: Gestione dispositivi mobile
- **Azure Sentinel**: Integrazione SIEM
- E altro...

---

## Risorse

- **[Guida Sicurezza Cloud Azure](https://documentation.wazuh.com/current/cloud-security/azure/index.html)** - Guida ufficiale
- **[Setup Integrazione Azure](https://documentation.wazuh.com/current/cloud-security/azure/index.html)** - Istruzioni step-by-step

---

## Setup Rapido

1. Crea service principal Azure
2. Concedi permessi richiesti
3. Configura Wazuh con credenziali
4. Monitora Log Analytics
5. Integra con Sentinel
