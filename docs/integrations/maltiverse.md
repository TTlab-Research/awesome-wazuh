# Guida Integrazione Maltiverse

Arricchisci l'intelligence minacce con feed minacce Maltiverse.

---

## Panoramica

Maltiverse fornisce dati di intelligence minacce per identificare IP malevoli, domini e file.

---

## Risorse

- **[Documentazione Integrazione Maltiverse](https://documentation.wazuh.com/current/user-manual/manager/integration-with-external-apis.html)** - Ufficiale
- **[API Maltiverse](https://www.maltiverse.com/)** - Documentazione API

---

## Setup

1. Ottieni chiave API Maltiverse
2. Configura in Wazuh
3. Abilita arricchimento intelligence minacce
4. Crea avvisi basati su indicatori abbinati
