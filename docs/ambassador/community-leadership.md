# Leadership della Community

Opportunità di leadership nella community Wazuh.

---

## Organizzazione Meetup Locale

**Passaggi**:
1. Crea gruppo meetup (meetup.com)
2. Imposta programma mensile
3. Trova speaker (ruota)
4. Promuovi alla community locale
5. Segnala presenze a Wazuh

**Supporto Wazuh**:
- Licenze gratuite per organizzatori
- Materiali promozionali
- Referral speaker
- Budget venue (talvolta)

---

## Gestione Community Online

**Canali**:
- GitHub Discussions
- Wazuh Slack
- Community Reddit
- Server Discord

**Responsabilità**:
- Rispondi alle domande
- Modera discussioni
- Segnala problemi/bug
- Condividi contenuti utili

---

## Traccia il Tuo Impatto

**Metriche Trimestrali**:
- [ ] Contenuti pubblicati
- [ ] Engagement community (reazioni, commenti)
- [ ] Eventi frequentati/organizzati
- [ ] Certificazioni rilasciate
- [ ] Referral generati
- [ ] Ricavi generati

**Obiettivi Annuali**:
- Contenuti: 12+ pezzi
- Speaking: 4+ eventi
- Referral: 5-10 organizzazioni
- Ricavi: $10-50k
